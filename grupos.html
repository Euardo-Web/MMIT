Como usar:

Para enviar mensagens, primeiro crie e configure uma instância na página de Gerenciar Instâncias. Depois, use os endpoints da API Z-API diretamente ou utilize esta interface para testes.

Instruções da API Z-API

api Endpoints Disponíveis

  • POST /api/instances - Criar nova instância 
    { "name": "Minha Instância", "contacts": ["contato1"], "message": "msg", "provider": "zapi" }
  • POST /api/instances/:id/start - Iniciar instância
  • GET /api/instances/:id/qr - Obter QR Code
  • GET /api/instances/:id/status - Verificar status
  • POST /api/instances/:id/send - Enviar mensagem 
    { "to": "5511999999999", "message": "Olá!" }
  • POST /api/instances/:id/stop - Parar instância
